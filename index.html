Terms and Condition Summariser

  • Designed and implemented a SaaS - Terms and Condition summariser built on pre-trained NLP model(legal-pegasus) and KeyBert library.
  • Working: The text is scraped from the files and undergoes segmentation for easier processing via threading. The summariser model takes in each segmented text as input and generates a summary. In addition to that, KeyBert library helps extract keyword from the generated summary for easy understanding. The segemented summaries are then compiled into one and written into a PDF file. The text is displayed on the website along with an option to download it as PDF. A feedback is also available which stores any feedback in a JSON file.
  • Built a frontend using HTML, CSS, & JS with flask as backend, where the model model was deployed.
  • Features: Custom Summary length, Flexible file input, Structured PDF Output, Feedback form

Early Prediction of Chronic Kidney Disease using Machine Learning

  • Designed and implemented a predictive learning machine learning model that analysed medical records to identify chronic kidney disease, achieving an accuracy of 93.33% and a recall score of 94.44%.
  • Four models were considered - Random Forest, Decision Tree, Logistic Regression, and XGBoost. Out of the 4, the overall performance of XGBoost was relatively better than the other 3.
  • Trained on a CKD dataset acquired from UC Irvine Machine Learning Repository comprising 400 records and synthetic data generated using Copulas library comprising 200 records.
  • Deployed the model locally via Flask with a user-friendly web interface scalable for public deployment using PythonAnywhere, enhancing accessibility and potential for broader user engagement.
